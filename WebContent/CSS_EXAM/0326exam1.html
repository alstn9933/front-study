font-family 속성 : 텍스트의 글꼴을 지정할 때 사용한다.

선택자 { font-family : 글꼴이름; }의 형식으로 사용된다.

font-size 속성 : 텍스트의 크기를 변경할 때 사용한다.

선택자 { font-size : 크기; }의 형식으로 사용된다.
  • 고정크기 px 테스트
  • 가변크기 % 테스트

font-weight 속성 : 두꺼운 글씨를 표현할 때 사용한다.

선택자 { font-weight : normal | bold | bolder | lighter | 100 | 400 | 900; }의 형식으로 표현한다.
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 가늘게
  • 원래 굵기보다 더 굵게

font-variant속성 : 작은 대문자로 변경할 때 사용한다.

선택자 { font-variant : 속성값; }의 형식으로 사용된다.

font-style 속성 : 텍스트의 기울임을 지정할 때 사용한다.

선택자 { font-style : normal | italic | oblique; }의 형식으로 사용한다.
  • italic 기울임 글꼴
  • oblique 기울임 글꼴

font 속성 : 글꼴 관연 스타일을 한번에 지정할 때 사용한다.

선택자 { font : font-style font-variant font-weight font-sizw/line-height font-family; }의 형식으로 사용한다.
  • 글꼴 관련 스타일 테스트1(16px 궁서체)
  • 글꼴 관연 스타일 테스트2(italic bold 20px 고딕체)
